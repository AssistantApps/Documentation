# App Review Badge

> Used to display the review details of an AssistantApps app.

Available from: [github.com/AssistantApps/WebComponents](https://github.com/AssistantApps/WebComponents)

### Params
 - apptype: `string`
 - platform: `string`

### Example

```html
  <assistant-apps-review-badge
    apptype="NMS"
    platform="1"
  />
```
